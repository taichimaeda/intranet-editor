# Intranet Editor

Edit your Intranet as you like (only compatible with Google Chrome).

![Screenshot 2021-03-16 at 17 27 45](https://user-images.githubusercontent.com/28210288/111344743-f33f2300-867c-11eb-8c5c-e9f5aa5571df.png)

## Installation Guide

1. Download the ZIP file from [this link](https://github.com/taichimaeda/intranet-editor/releases/download/v1.0/intranet-editor.zip).
2. Extract the ZIP file by double-clicking it.
3. Follow the steps in [this video](https://www.youtube.com/watch?v=19UZcGczmbA) to install the extension via developper mode.
